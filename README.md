
## Night-Breeze


### 快速启动
1. 包安装
```
npm i 
```

2. 在src/config 文件夹下面创建.envConfig.json文件可覆盖环境变量
```
cp .\src\config\.envConfig.example.json .\src\config\.envConfig.json
```
3. 运行项目
```
npm run start
```
4. 文档地址：http://localhost:3000/document/index.html
